# TeamFinder

Платформа для поиска команды на pet-проекты. Реализована базовая функциональность из задания и вариант 2: навыки пользователей с добавлением без перезагрузки страницы и фильтрацией участников по навыкам.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver --insecure
```

Сайт откроется по адресу `http://127.0.0.1:8000/`.

Для заполнения демо-данными:

```bash
python manage.py seed_demo
```

Логины: `anna@yandex.com`, `igor@yandex.com`, `maria@yandex.com`. Пароль для всех: `testpass123`.

## Запуск через Docker Compose

```bash
docker compose up --build
```

В этом режиме используется PostgreSQL и volume `postgres_data`.

## Проверка

```bash
python manage.py test
```

## Что есть в проекте

- регистрация и вход по email;
- публичные профили пользователей;
- редактирование профиля владельцем;
- создание, редактирование и завершение своих проектов;
- присоединение к открытым проектам;
- список пользователей и список проектов с пагинацией по 12 элементов;
- навыки в профиле с автодополнением и созданием новых навыков;
- фильтр пользователей по выбранному навыку;
- смена пароля для авторизованных пользователей;
- админ-панель для управления пользователями, профилями, проектами и навыками.
